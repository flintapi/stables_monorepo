## Transaction request

Find a transaction by its `id` or `reference` using the `<BASE_URL>/v1/ramp/transactions/` endpoint. Calling the endpoint without any query parameters will return a list of all transactions for the organisation.

```sh tab="Get all transactions"
curl --request GET \
  --url https://stables.flintapi.io/v1/ramp/transactions \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: <FLINT_API_KEY>'
```

```sh tab="Get with id"
curl --request GET \
  --url https://stables.flintapi.io/v1/ramp/transactions?id={TRANSACTION_ID} \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: <FLINT_API_KEY>'
```

```sh tab="Get with reference"
curl --request GET \
  --url https://stables.flintapi.io/v1/ramp/transactions?reference={TRANSACTION_REFERENCE} \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: <FLINT_API_KEY>'
```
