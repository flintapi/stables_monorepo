# Packages folder for flintapi monrepo

This the packages folder that holds utility, shared and core packages for the monorepo

## Utility Packages

- shared - contains shared types, utils, adapters, middleware, telemetry, helpers, observers, validators
- core - conatins system operation services like, qstash (client) job creation and dispatcher, redis client, service registry
- db - contains db orchestration, migrations and seeders, connection utilities
